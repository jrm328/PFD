1. Run the Game.
	a) Open GameplayCargoCourier scene in Scenes folder.
	b) Press Play button.
	c) Use LMB(pc) or touch(mobile) to control the Player.

2. Game Settings.
	a) All the game settings are at: Hierarchy Tab => ApplicationContext => Settings.
	b) To open Cheat Panel:
		1) Hierarchy Tab => ApplicationContext => Settings => Show Cheat Panel check box.
		2) Press Play button. Touch with 3 fingers(mobile)/LMB click(pc) and hold for 3 sec.

3. Game Balance.
	a) Game Balance can be adjusted at: Hierarchy Tab => ApplicationContext => GameModel => BalanceModel.

4. Future Updates(will be added in case of interest in this game)
	a) Traveling to another cities.
	b) Enemies.
	c) Train Parcel.
